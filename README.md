## AdventOfCode

Repo that contains all the tasks for that competition. The repo has the following structure:

```
AdventOfCode/
├── data/
│   ├── input_exercise1.txt
│   ├── input_exercise2.txt
│   └── ...
├── days/
│   ├── exercise1.py
│   ├── exercise2.py
│   └── ...
└── tests/
    ├── test1.py
    ├── test2.py
    └── ...
```

Where for any exercise you should just run:

```
python days/exercise3.py
```
